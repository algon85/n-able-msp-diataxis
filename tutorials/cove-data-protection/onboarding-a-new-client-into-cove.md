## Purpose
Provide a step-by-step guide for onboarding a new client into Cove Data Protection, from initial setup through first successful backup.

## Prerequisites
- Client exists in N-central or N-sight  
- Administrative access to Cove Data Protection  
- Agent deployment method identified  

## Procedure
1. Create or verify the client in Cove  
2. Configure default backup policies  
3. Deploy Cove agents to endpoints/servers  
4. Verify initial backup success  

## Validation
- Backup jobs report success  
- Devices appear correctly under the client  
- Restore test (file or folder) completed
