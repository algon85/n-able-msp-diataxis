# Cove Data Protection Tutorials

This section contains learning-oriented tutorials for **Cove Data Protection**, focused on helping MSP staff understand and use the platform through guided workflows.

## Audience
These tutorials are intended for:
- Helpdesk analysts and NOC professionals
- MSP staff managing backup and recovery
- Technicians new to the Cove platform

## Scope
The tutorials in this directory focus on:
- Core backup and recovery workflows
- Platform setup and basic operations
- Monitoring and verification of backups

They do **not** cover advanced configuration, scripting, or enterprise automation.

## How to Use These Tutorials
Each tutorial is designed to be followed sequentially and assumes minimal prior experience with Cove Data Protection.
Thus, readers may wish to review the relevant materials in the [Explanations](../../explanations/) and [Reference](../../reference/) sections first.
Readers should treat these as exercises to understand workflows rather than exhaustive operational references.
