# Monitoring and Alerting Basics (N-central)

**Status:** Draft  
**Diataxis Type:** Tutorial  
**Product:** N-central RMM

## Purpose
This tutorial provides an intro to basic monitoring and alerting concepts in N-central.
The intention is to understand how monitoring policies and alerts work together to provide visibility into endpoint health.

## Scope
- Assumes agents are already deployed and checking in
- Focuses on foundational concepts rather than advanced tuning
- Does not cover alert noise reduction or escalation workflows

## Prerequisites
- Active N-central agents reporting to the system
- Access to monitoring and alert configuration features

## Outline
- What monitoring and alerting mean in N-central
- Monitoring policies and checks
- Alert severity and notification concepts
- Applying basic monitoring to endpoints
- Confirming alerts are generated and received
