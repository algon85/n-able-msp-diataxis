# N-central Tutorials

This section contains learning-oriented tutorials for **N-central RMM**, focused on helping MSP staff understand and use the platform through guided workflows.

## Audience
These tutorials are intended for:
- Helpdesk and NOC engineers
- MSP staff managing endpoints and monitoring
- Technicians new to N-central

## Scope
The tutorials in this directory focus on:
- Core workflows like client onboarding, agent deployment, and monitoring
- Platform setup and initial configuration
- Best practices for common operational tasks

They do **not** cover advanced automation, enterprise-scale customization, or scripting.

## How to Use These Tutorials
Each tutorial is designed to be followed sequentially. For conceptual background or terminology, readers may wish to review the relevant materials in the [Explanations](../../explanations/) and [Reference](../../reference/) sections first.
