# Deploying Agents at Scale (N-central)

**Status:** Draft  
**Diataxis Type:** Tutorial  
**Product:** N-central RMM

## Purpose
This tutorial introduces the workflow for deploying N-central agents across multiple endpoints.
The goal is to describe the *process* of large-scale agent deployment.

## Scope
- Assumes an existing N-central instance
- Assumes clients and sites already exist
- Focuses on deployment strategy and verification
- Does not cover client onboarding or monitoring design

## Prerequisites
- Admin access to N-central
- Appropriate permissions on target endpoints
- Network connectivity between endpoints and N-central

## Outline
- Deployment prerequisites and planning
- Common agent deployment methods
- Preparing agent installers
- Deploying agents to multiple endpoints
- Verifying agent installation and check-in
