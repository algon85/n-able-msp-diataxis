# N-sight RMM Tutorials

This section contains learning-oriented tutorials for **N-sight RMM**, focused on helping technicians understand and use the platform through guided workflows.

## Audience
These tutorials are intended for:
- Helpdesk and junior technicians
- Users new to the N-sight platform
- MSP staff transitioning from other RMM tools

## Scope
The tutorials in this directory focus on:
- Core N-sight workflows
- Simplicity and ease of deployment
- Small to mid-sized MSP use cases

They do **not** attempt to cover advanced customization or large-scale enterprise scenarios.

## How to Use These Tutorials
Each tutorial is designed to be followed in sequence and assumes minimal prior knowledge of N-sight.
Readers should treat these as learning exercises rather than exhaustive operational references.
