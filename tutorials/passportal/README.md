# Passportal Tutorials

This section contains learning-oriented tutorials for **Passportal**, the password management and documentation platform used by MSPs.

## Audience
These tutorials are intended for:
- Helpdesk and NOC professionals
- MSP staff managing credentials and documentation
- Technicians new to Passportal

## Scope
The tutorials in this directory focus on:
- Core credential management workflows
- Onboarding clients and adding devices
- Secure sharing and auditing

They do **not** cover advanced integrations, scripting, or enterprise-scale customization.

## How to Use These Tutorials
Each tutorial is designed to be followed step-by-step and assumes minimal prior knowledge of Passportal.
Thus, readers may wish to review the relevant materials in the [Explanations](../../explanations/) and [Reference](../../reference/) sections first.
Readers should treat these as exercises to understand workflows rather than exhaustive operational references.
