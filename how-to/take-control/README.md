# Take Control How-Tos

This section contains task-oriented guides for **Take Control**, N-able's remote access platform.

## Audience
These how-tos are intended for:
- Helpdesk and support analysts
- MSP staff performing remote support sessions
- Technicians familiar with Take Control basics

## Scope
The how-tos in this directory focus on:
- Quick, step-by-step instructions for common remote access tasks
- Troubleshooting remote sessions
- Safe access and operational best practices

They do **not** cover onboarding or advanced integrations.

## How to Use These How-Tos
Each guide assumes the reader has basic familiarity with Take Control workflows. For background concepts, see the [Explanations](../../explanations/) and [Reference](../../reference/) sections.
