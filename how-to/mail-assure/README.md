# Mail Assure How-Tos

This section contains practical guides for **Mail Assure**, N-able's email security and management platform.

## Audience
These how-tos are intended for:
- Helpdesk and support analysts
- MSP staff managing email security and quarantine
- Technicians familiar with Mail Assure concepts

## Scope
The how-tos in this directory focus on:
- Spam and quarantine management
- Quick, task-based instructions
- Resolving common email security issues

They do **not** cover advanced integrations or platform-wide configuration.

## How to Use These How-Tos
Each guide assumes familiarity with Mail Assure basics. For background concepts, see the [Explanations](../../explanations/) and [Reference](../../reference/) sections.
