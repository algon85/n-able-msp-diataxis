# N-central How-Tos

This section contains practical, task-focused guides for **N-central RMM**, designed to help technicians solve common operational problems.

## Audience
These how-tos are intended for:
- Helpdesk and NOC professionals
- MSP staff familiar with N-central workflows
- Technicians who need fast, actionable instructions

## Scope
The how-tos in this directory focus on:
- Solving common issues with agents, alerts, and commands
- Short, step-by-step instructions
- Troubleshooting everyday operational problems

They do **not** teach the full platform or cover advanced configurations.

## How to Use These How-Tos
Each guide assumes basic familiarity with N-central concepts. For conceptual background or terminology, see the [Explanations](../../explanations/) and [Reference](../../reference/) sections.
